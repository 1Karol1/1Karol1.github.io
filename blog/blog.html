Today I will explain the differences between relative, absolute, and fixed positioning.

Position absolute and relative:

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed). However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.

This div element has position: absolute;

Position fixed:

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element. Notice the fixed element in the lower-right corner of the page.
