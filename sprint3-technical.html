CSS vs HTML

I would describe the difference between CCS and HTML on an example of a house. When you start to build it you decide about general layout, dimmensions, windows and doors position, roof type etc. That would be your HTML code. Once you have a general construction you start to finish your design by colouring outside, inside, decorating interior, grow a lawn and some plants around it. Position your furniture inside it. So now you are using CSS. As you can see HTML + CSS are nescesary to completly build a website like all these factors are to build a house

Loops and control flow

Loops and contol flow are working together to do certain tasks corectly. Control flow means that we do certain things in specific order. Just as we do driving a car. We put the seatbelt on first. Than we start the engine, releasing hand brake, selecting gear, releasing foot brake and than accelerate. If we do these steps in wrong order we will simply not go anywgere and there is a risk of crashing too!

Loops are like driving to work. We drive the same route every day to and from work. Except on weekends when we don't need to go there. That is the condition under the loop is running. Run the loop Monday to Friday, don't run the loop on Saturdays, Sundays, public holidays and leave.

DOM

Dom in my native language means a house. Again that example seems to be quite accurate here. Imagine Document Object Model as a house. As we said before HTML is a structure, CSS is looks and everything which is a part of our house can be changed by us. Just like in DOM we can manipulate elements like titles, paragraphs, we can change the looks, change layout of the website. It is very useful if we want to see how changes in code will be visible on the screen.

Data access form arrays and objects

Because arrays and objects have differnt structure we access data in different ways. If we would like to extract data from the array we have to point the element which we would like to access. In case of arrays nested inside other array like [[a,b][c,d][e,f]] we must first point out the internal array position and than point the element we want to access.

In case of objects which contain some properties we must ask which property we would like to access. For example an object of a car will have properties like colour, make, model, engine size, doors number, year, value. If we ask for certain parameter we will get a value assigned to that parameter.

Functions
